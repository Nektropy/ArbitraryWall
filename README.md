# 🧱 便签投稿说明

欢迎向便签墙投稿！每一张便签都是你留在墙上的印记。

---

## 📦 投稿格式

投稿内容是一段 JSON 对象，发给站长即可，他会帮你贴到墙上。

```json
{
  "text": "你的便签内容（支持 Markdown）",
  "style": "便签颜色或背景图片",
  "author": "你的昵称（可选）"
}
```

---

## ✏️ text — 便签内容

便签支持 **Markdown 语法**，可以加粗、斜体、列表、链接等。

### 📏 字数限制：100 字

> 字数按**可见内容**计算，Markdown 符号（`**`、`#`、`-`）和链接地址 `(url)` **不占字数**。
> 超出 100 字会自动截断，建议提前用底部的[检测工具](#-检测工具)确认。

### 支持的语法示例

| 效果 | 写法 |
|------|------|
| **粗体** | `**粗体**` |
| *斜体* | `*斜体*` |
| ~~删除线~~ | `~~删除线~~` |
| 标题 | `### 标题` |
| 无序列表 | `- 项目一` |
| 有序列表 | `1. 第一项` |
| 引用 | `> 引用内容` |
| 链接 | `[显示文字](https://example.com)` |
| 行内代码 | `` `代码` `` |
| 代码块 | ` ```代码块``` ` |
| 分隔线 | `---` |

### 示例

```
**今天心情很好** 🌞
- 吃了好吃的拉面
- 见到了许久不见的老朋友
> 生活就是这些小确幸
[我的博客](https://example.com)
```

## 🎨 style — 便签样式

`style` 支持三种写法：

### 1. 预设颜色名

直接填颜色名字（英文小写）：

| 颜色名 | 效果 |
|--------|------|
| `yellow` | 🟡 经典黄（默认） |
| `blue` | 🔵 天空蓝 |
| `green` | 🟢 清新绿 |
| `pink` | 🩷 少女粉 |
| `red` | 🔴 热情红 |
| `purple` | 🟣 梦幻紫 |
| `orange` | 🟠 活力橙 |
| `teal` | 🩵 薄荷青 |
| `indigo` | 💙 靛蓝 |
| `lime` | 💚 柠檬绿 |
| `cyan` | 🩵 清澈青 |
| `amber` | 🟡 琥珀金 |

```json
"style": "pink"
```

### 2. 自定义颜色（十六进制）

填写以 `#` 开头的颜色值：

```json
"style": "#a8d8ea"
```

可以用 [ColorPicker](https://htmlcolorcodes.com/) 选取你喜欢的颜色。

### 3. 图片背景

填写以 `http` 开头的图片链接，便签将以该图片为背景（内容区会加半透明白底）：

```json
"style": "https://images.unsplash.com/photo-xxxxxx?w=300&h=200"
```

> 💡 推荐使用 [Unsplash](https://unsplash.com)、[Imgur](https://imgur.com) 等稳定的图床，避免图片失效。
> 建议图片比例约 **3:2**，尺寸 300×200px 左右。

---

## 👤 author — 署名（可选）

你的昵称，会显示在便签右下角：`— 你的名字`

```json
"author": "向日葵"
```

不填则便签不显示署名。

## 📋 完整示例

### 示例一：普通文字便签
```json
{
  "text": "**今天** 看了一部很好看的电影\n> 生活需要一点仪式感",
  "style": "blue",
  "author": "影迷小王"
}
```

### 示例二：带链接的便签
```json
{
  "text": "我的新文章写好啦！\n[点击阅读](https://myblog.com/post/1)",
  "style": "green",
  "author": "写字的人"
}
```

### 示例三：图片背景便签
```json
{
  "text": "🌸 春天来了",
  "style": "https://images.unsplash.com/photo-1522748906645-95d8adfd52c7?w=300",
  "author": "春日限定"
}
```

### 示例四：自定义颜色，无署名
```json
{
  "text": "匿名留言一条～",
  "style": "#ffe4b5"
}
```

## ⚙️ 检测工具

在投稿前，建议用配套的 **`便签检测工具.html`** 检查：
- 当前字数是否超出 100 字限制
- 便签的实际显示效果预览

用浏览器打开 `便签检测工具.html` 即可使用，无需联网。

---

## 📮 如何提交

把你的 JSON 内容发送给站长即可，可以通过以下方式：
- 私信 / 评论区留言
- 发送到指定邮箱

站长会审核后手动添加到墙上。

---

*字数统计逻辑：Markdown 语法符号（`**`、`#`、`-`、`>`等）和链接中的 URL 部分不计入字数，只统计实际可见文字内容。*
